# Product Vision

This app is a Chia/Base bridge-style web app.

Goal:
Make it extremely easy for a user to pay USDC on Base and receive XCH to their Chia wallet.

Target user:
Someone who wants XCH but does not already have XCH in their wallet.

Must feel:
Simple, trustworthy, fast, safer than sketchy bridge sites.

Core flow:
1. User connects EVM wallet
2. User enters Chia receive address
3. User chooses USDC amount
4. App shows exchange rate, fees, and estimated XCH
5. User confirms
6. App tracks order status
7. User receives XCH
